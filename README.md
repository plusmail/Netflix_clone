## 넷플릭스 클론

프론트 엔드 React & Redux 공부용 Neflix Clone App.
MovieDB Api 를 사용해서 영화를 검색하고 세부 정보를 확인할 수 있습니다.



### Learn
+ React
+ Redux thunk
+ Middleware
+ API

### Tools
+ Axios
+ React & Redux
+ Sass (grid & flexbox)
+ Media queries

### Skills
+ 영화 검색 기능
+ 매주 업데이트 되는 데이터를 통해 트렌드 영화를 볼 수 있다.
+ 사용자는 영화를 클릭할 수 있고 모달도 뜬다.
+ 반응형 적용
